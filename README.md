# InfoBooks

Создать проект C# ASP.NET MVC с Web API (не Net Core).
В нем создать сущностную модель данных Entity Framework для базы книг.
Для каждой сущности создается свой свой контроллер Web API. Хорошо бы, чтобы контроллер не работал напрямую с БД, желательны классы посредники - репозитории.
Приложение должно быть SPA, то есть одностраничное, без перезагрузки страницы при переходе между экранами.
Все данные запрашиваются через Web Api на стороне сервера и с помощью angularJS методов на стороне клиента,
Работа с базой данных осуществляется с помощью Entity Framework и LINQ.

Всего три экрана.
1. Стартовый - по сути пустой, с картинкой на фоне.
2. Список авторов
3. Список книг (автор, книга, год, цена)
